OpenStruct implement for RubyMotion

Usage
---
```
v = Motion::OpenStruct.new(:one => 'two')
p = Motion::OpenStruct.new(:one => 'two')
puts v
puts v == p
```

Installation
---
1. Add this to your `Gemfile`: `gem 'motion-ostruct'`
2. Run `bundle install`
